PMDCI'S FLAGS PACK

Version 2.1.1 [2012-Jun-25]
By Pedro Innecco

 

ABOUT

These flag icons has been compiled and formated by Pedro Innecco (that's me!). I am a flag-enthusiast (a hobbyist vexillologist, the one who study flags) and I've been studying flags since I was four years old. I guess the colourful pictures just captivated me as a child.

This pack contain the flags of all countries, territories, or areas of geographical interest that meet the criteria for inclusion in the ISO ISO3166-1 standard.

USAGE

These flags were designed to be used within applications as a way to represent countries and regions. For example, upon selecting a country/region from a drop-down list, in reports and dashboards, tables which contain columns depicting a location (e.g.: a user list in a community portal, or in user profile pages), and so on. This pack has been used with several different applications, such as:

  • Microsoft Dynamics CRM 4.0, 2011
  • Microsoft SharePoint 2007, 2010
  • Blogs and Content Management Systems such as Drupal, Joomla, Xaraya and Wordpress

POLITICS

If you feel that a given flag is not appropriate, feel free to replace it by following the guidelines below. But please bear in mind that the author is not interested to engage in political debates and would like to stay as neutral as possible about flags and territorial disputes. Moreover, if you would like to make any POLITE suggestions, please contact the author directly.

FLAGS GUIDELINES

  • IMAGE SIZE: All flags must follow their correspondent official scale, and be 16 pixels in height.
  • IMAGE FORMAT: All flags must be on PNG format.
  • NOMENCLATURE: All files are originally named as flag_xx.png, where xx is the ISO3166-1 Alpha-2 code for the country or region. For example, flag_ch.png for Switzerland and flag_br.png for Brazil and so on. Additionally, a file called flag_aa.png is included so you can use for when there is no Country selected from a given Country/Region list. Starting with version 2.0, the pack includes a script file named FLAGPACK-MV.CMD that can be executed from the command prompt of any Microsoft Windows system (XP or superior) that will allow the renaming of flag icon files between the ISO3166-1 Alpha-2 code (two-letter country code) or the ISO3166-1 Numerical code.
